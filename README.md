# 📋 逐句提示词阅读器

> 逐句显示提示词的 PWA 提词器——粘贴台词，一句一句读，支持语音识别自动翻页，专为手机端打造，可离线使用。

---

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 📝 **多行提示词输入** | 粘贴或输入文本，每行视为一句提示词 |
| 🔄 **逐句显示** | 每次只显示当前一句，点击「我读完了」切换下一句 |
| 🎙️ **自动模式** | 开启后利用 Web Speech API 监听语音，说话停顿后自动翻到下一句 |
| 📊 **进度条** | 实时显示当前朗读进度（第 N / 共 M 句） |
| 💾 **本地持久化** | 提示词内容自动保存到 `localStorage`，刷新后不丢失 |
| 📵 **离线支持** | 通过 Service Worker 缓存所有静态资源，断网仍可正常使用 |
| 📱 **手机优先** | 大字体、大按钮、深色主题，专为手机端台词提词场景设计 |
| ⬇️ **可安装为 App** | 符合 PWA 标准，可添加到手机主屏幕，获得原生 App 体验 |

---

## 🚀 快速上手

### 在线使用

直接用手机或桌面浏览器打开部署地址即可，无需安装任何软件。

### 本地运行

项目为纯静态页面，无需构建步骤，任意静态文件服务器均可使用：

```bash
# 方式一：使用 Python 内置服务器
python3 -m http.server 8080

# 方式二：使用 Node.js npx serve
npx serve .

# 方式三：使用 VS Code Live Server 插件直接打开 index.html
```

打开浏览器访问 `http://localhost:8080` 即可。

> ⚠️ **注意**：自动模式（语音识别）和 Service Worker 需要在 **HTTPS** 或 `localhost` 环境下才能正常工作。

## 📖 使用说明

### 第一步：输入提示词

1. 点击右上角 **✏️ 编辑** 按钮，打开输入面板。
2. 在文本框中粘贴或输入提示词，**每行一句话**。
3. 点击 **💾 保存** 按钮，提示词将被解析并保存。

```
大家好，我是主持人小明。
今天我们来聊聊人工智能的未来。
让我们一起期待美好的明天！
```

### 第二步：开始朗读

- 页面中央的大卡片显示**当前需要朗读的句子**。
- 朗读完毕后，点击 **✅ 我读完了** 按钮切换到下一句。
- 点击 **🔄 重置** 按钮可随时回到第一句。

### 第三步（可选）：开启自动模式

- 在底部开启 **🎙️ 自动模式** 开关。
- 浏览器会请求**麦克风权限**，请点击允许。
- 之后只需朗读，说完停顿后会**自动翻到下一句**，无需手动点击。
- 卡片边框会出现蓝色发光动效，并在右下角显示「识别中」指示器，表示正在监听。

### 安装为手机 App

1. 用 Chrome（Android）或 Safari（iOS）打开应用。
2. 点击浏览器菜单中的 **"添加到主屏幕"**。
3. 图标将出现在手机桌面，点击即可全屏启动，获得原生 App 体验。

## 🗂️ 项目结构

```
line-by-line-prompt/
├── index.html      # 主页面（HTML 结构）
├── style.css       # 样式（深色主题，手机适配）
├── app.js          # 主逻辑（提示词解析、语音识别、UI 交互）

├── manifest.json   # PWA Manifest（App 名称、图标、主题色）
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

## 🛠️ 技术栈

- **纯原生 Web**：HTML5 + CSS3 + Vanilla JavaScript，零依赖，零构建
- **PWA**：`manifest.json` + Service Worker，支持离线访问与主屏安装
- **Web Speech API**：`SpeechRecognition`，实现浏览器端语音识别
- **localStorage**：提示词内容持久化
